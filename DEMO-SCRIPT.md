## Production Notes

### Visual Requirements
- **High-quality screen recording** (1080p minimum)
- **Smooth transitions** between scenes
- **Clear cursor movements** and UI interactions
- **Readable text** on screen elements
- **Professional audio** quality

### Technical Setup
- **Stable demo environment** with pre-loaded data
- **Fast internet** for smooth recording
- **Backup recordings** in case of technical issues
- **Multiple takes** for best quality

### Editing Guidelines
- **Add captions** for accessibility
- **Include background music** (subtle, professional)
- **Speed up repetitive sections** (maintain clarity)
- **Highlight key metrics** with visual emphasis
- **Add zoom effects** for important details

### Content Optimization
- **Keep energy high** throughout demo
- **Speak clearly and confidently**
- **Use hand gestures** (if visible)
- **Maintain eye contact** with camera
- **Show enthusiasm** for the technology

### Common Pitfalls to Avoid
- **Don't rush through** technical details
- **Don't use jargon** without explanation
- **Don't show loading screens** unnecessarily
- **Don't have dead air** time
- **Don't forget to mention** GLM 5.1 integration

### Success Metrics
- **Viewer retention** through entire video
- **Clear understanding** of value proposition
- **Technical credibility** established
- **Emotional connection** with problem/solution
- **Call-to-action** effectiveness
